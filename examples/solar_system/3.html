Solar System 3
Use the 'd' key to toggle debugging information on and off.
Use the 'a' key to toggle antialiasing on and off.
Use the 'g' key to toggle gamma correction on and off.
Use the f/F keys to slow down or speed up the frame rate.
Use the 'p' key to toggle between parallel and orthographic projection.
Use the e/E keys to change the angle of the ecliptic plane.
